FAQ

Q. What is Steward?

Steward is a Kubernetes Operator that manages hosted control planes. It runs control planes as pods in a management cluster, making multi-tenant Kubernetes more cost-effective and easier to operate.

Q. How does Steward relate to Kamaji?

Steward is a community-governed fork of Kamaji by Clastix Labs, providing stable releases under the Apache 2.0 license.

Q. How does it work with Cluster API?

They complement each other: Steward simplifies control plane management, while Cluster API handles infrastructure abstraction and lifecycle management.
